Box Model

Kiekvienas HTML elementas turi dėžutės modelį, kuriame yra nurodoma informacija apie turinio, pamušalo, rėmelio, paraščių ir pozicijų dydžius.

  1. Content - turinys, centrinė dalis
  2. Padding - pamušalas, nurodo tarpą tarp turinio ir rėmelio
  3. Border - rėmelis, yra tarp padding ir margin
  4. Margin - paraštės, nurodo tarpą tarp rėmelio ir kitų aplinkinių elementų
  5. Positions - modifikuoja elementą poziciją
Box Model Pavyzdys

Content

Turinys, kuris yra tekstas / media, kurį galima įvairiai stilizuoti.

Padding

Padding / pamušalas - tarpas tarp turinio ir rėmelio.
Jį galima nustatyti 8 būdais.

  • padding-top: px;
  • padding-right: px;
  • padding-bottom: px;
  • padding-left: px;
  • padding: all;
  • padding: top_bottom right_left;
  • padding: top right_left bottom;
  • padding: top right bottom left;

Border

Rėmelis yra juosta atskirianti padding ir margin. Rėmelis taip pat nurodo ir fono ribas.

Border'iui galima suteikti storio, stiliaus, spalvos ir kampų apvalinimo stilius. Skirtingus stilius galima suteikti skirtingoms kraštinėms/kampams.

  • border: width style color;
  • border-width: px;
  • border-style: style;
  • border-color: color;
  • border-top/right/bottom/left: width style color;
  • border-top/right/bottom/left-color: color;
  • border-radius: 5px;
  • ...
  • ..
  • .

Margin

Margin top ir bottom neveikia linijiniams elementams.

Margin / paraštės - tarpas tarp rėmelio ir aplinkinių elementų.
Jį galima nustatyti 8 būdais.

  • margin-top: px;
  • margin-right: px;
  • margin-bottom: px;
  • margin-left: px;
  • margin: all;
  • margin: top_bottom right_left;
  • margin: top right_left bottom;
  • margin: top right bottom left;
