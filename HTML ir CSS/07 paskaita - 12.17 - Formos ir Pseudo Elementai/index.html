Form

Formos reikalingos norint pasiimti vartotojo įvestis. Plačiau apie tai JS temos metu.

Formos tag'ai

  • form - formos tag'as
  • label - formos įvesties etiketė/pavadinimas
  • input - formos įvesties laukas (įvairus)
  • textarea - formos įvesties laukas (dideliam teksto kiekiui)
  • select - opcijų pasirinkimo laukas (nedideliam kiekiui opcijų)
  • datalist - opcijų pasirinkimo laukas (dideliam kiekiui opcijų)
  • option - opcija
  • output - formos atsakymui išvesti
  • button - mygtukas (asmeniškai nesiūlau naudoti formose)

Formos tag'ų atributai

  • type - input - nurodo tipą
    • text - tekstas
    • password - slapyvardis (atvaizduojamas kaip taškeliai)
    • number - skaičius
    • email - elektroninis paštas
    • url - nuoroda
    • checkbox - daugelio pasirinkimas
    • radio - vieno pasirinkimas
    • submit - formos pateikimas
    • file - failo iš kompiuterio pasirinkimas
    • tel - telefonas
    • range - skaičiaus pasirinkimas slenkančioje juostoje
    • date - datos pasirinkimas (metų, mėnesio, dienos)
    • datetime-local - datos ir laiko pasirinkimas (metų, mėnesio, dienos, valandos, minutės)
    • month - mėnesio pasirinkimas
    • week - savaitės pasirinkimas
    • time - laiko pasirinkimas
    • color - spalvos pasirinkimas
    • button - mygtukas be funkcionalumo
    • image - paveiksliukas skirtas pateikti formą
    • hidden - paslėpta įvestis
    • reset - išvalo formos įvestis
    • search - paieškos laukas
  • for - label - nurodo kuriai įvesčiai priskirti label'į
  • id - įvestims - apjungia label'į su įvestimi
  • name - įvestims - suteikia įvesčiai vardą (susigalvotą) (dažnu atveju toks pats kaip ir id)
  • value - įvestims - įvesties reikšmė
  • required - įvestims - privaloma įvestis
  • placeholder - input+textarea - pavyzdinis tekstas
  • min - input(number/range/date) - minimali įvestis
  • max - input(number/range/date) - maksimali įvestis
  • step - input(number/range) - kitimo žingsnis
  • accept - input(file) - kokio tipo failai leistini
  • pattern - input(tel/...) - nurodo kokia turi būti įvesties seka (naudojant regex)
  • rows - textarea - eilučių kiekį (vizualus dydis)
  • cols - textarea - stuplelių kiekį (vizualus dydis)
  • checked - input(checkbox/radio) - iš anksto pažymėtas variantas
  • selected - option - iš anksto parinkta opcija
  • disabled - įvestims - išjungtas įvesties laukas
  • readonly - įvestims - neleidžiama keisti reikšmės
  • multiple - select - ledžia pasirinkti daug variantų
  • ...

Pseudo Class (Form)

  • :checked - pažymėtas(checkbox/radio)
  • :default - pradinis (kur pradžioje buvo: checked arba selected)
  • :disabled - turi disabled atributą
  • :enabled - neturi disabled atributo
  • :focus - šiuo metu fokusuojama įvestis
  • :in-range - min-max ribose
  • :out-of-range - už min-max ribų
  • :indeterminate - neapibrėžta reikšmė (nesvarbu)
  • :valid - tinkama reikšmė
  • :invalid - netinkama reikšmė
  • :required - privaloma įvestis (turi required atributą)
  • :optional - neprivaloma įvestis (neturi required atributo)
  • :readonly - elementas turi readonly atributą
  • :read-write - elementas neturi readonly atributo
