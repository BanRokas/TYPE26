Display Grid

Parent (Container)

  • display: grid;
  • grid-template-rows: fr px %;
  • grid-template-columns: for px %;
  • grid-template: grid-template-rows / grid-template-columns;
  • grid-template-areas: ". . ." ;
  • gap: row-gap column-gap;
  • align-items: start | end | center | stretch;
  • justify-items: start | end | center | stretch;
  • place-items: align-items justify-items;
  • align-content: start | end | center | stretch | space-between | space-around | space-evenly;
  • justify-content: start | end | center | stretch | space-between | space-around | space-evenly;
  • place-content: align-content justify-content;
  • grid-auto-rows: fr px %;
  • grid-auto-columns: fr px %;
  • grid-auto-flow: row(default) | column | dense | row dense | column dense;
  • grid: grid-template-rows grid-template-columns grid-template-areas grid-auto-rows grid-auto-columns grid-auto-flow;

Children (Items)

  • grid-row-start: number;
  • grid-column-start: number;
  • grid-row-end: number;
  • grid-column-end: number;
  • grid-row: grid-row-start / grid-row-end;
  • grid-column: grid-column-start / grid-column-end;
  • grid-area: name | grid-row-start / grid-column-start / grid-row-end / grid-column-end;
  • z-index: number; (nereikia pozicijos jeigu grid vaikinis)
  • align-self: start | end | center | stretch | baseline;
  • justify-self: start | end | center | stretch | baseline;
  • place-self: align-self justify-self;

Units and Functions

  • fr - frakcija
  • min-content
  • max-content
  • auto
  • fit-content()
  • minmax(min, max)
  • min(value1, value2, value3...)
  • max(value1, value2, value3...)
  • repeat(kiek, koki/ius)
  • subgrid
