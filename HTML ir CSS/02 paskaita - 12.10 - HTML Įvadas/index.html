Tag'ų sąrašas (semantikai)

  • html - apgaubiantis visą HTML kodą
  • head - puslapio informacija
  • meta - nurodo meta informaciją
  • link - prikabina kitus failus
  • title - nurodo tab'o pavadinimą
  • body - apgaubia vartotojui atvaizduojamą turinį
  • header - navigacija/menu/prisijungimai...
  • main - puslapio informacija
  • footer - copyrights, partners, contacts, cookies, terms and licenses...

Tag'ų sąrašas (turiniui)

  • h1-h6 - heading - antraštės
  • p - paragraph - paragrafas
  • a - anchor - nuoroda
  • img - image - paveiksliukas
  • ol - ordered list - rikiuotas sąrašas
  • ul - unordered list - nerikiuotas sąrašas
  • li - list item - sąrašo elementas
  • i - italic - pakreipia tekstą
  • em - emphasis - pakreipia tekstą ir nurodo jį kaip svarbesnį
  • b - bold - paryškina tekstą
  • strong - strong - paryškina tekstą ir nurodo jį kaip svarbesnį
  • span - span - skirtas išskirti teksto gabaliuką ir kažkaip sustilizuoti
  • br - line break - eilutės lūžis
  • hr - thematic break - eilutės lūžis su brūkšniu
  • button - button - mygtukas
  • form - form - forma
  • label - label - įvesties įvardinimas
  • input - input - įvestis (gali būti labai įvairi)
  • textarea - text area - didelė tekstinė įvestis
  • select - select - varianto pasirinkimas
  • option - option - variantas
  • fieldset - field set - lauko apipavidalinimas
  • legend - legend - lauko pavadinimas
  • table - table - lentelė
  • thead - table head - stulpelių pavadinimų skiltis
  • tbody - table body - duomenų skiltis
  • tfoot - table foot - apibendrinimo skiltis
  • tr - table row - lentelės eilutė
  • td - table data - lentelės duomuo
  • th - table heading - lentelės stulpelio pavadinimas

Atributų sąrašas

  • href - hyperlink reference - nuoroda į kažkur (dažniausiai puslapį)
  • src - source - šaltinis/nuoroda į kažką (dažniausiai failą)
  • alt - alternative text - medijai skirtas alternatyvus tekstas
  • width - width - nurodo elemento plotį
  • height - height - nurodo elemento aukštį
  • type - input type - įvesties tipas (tekstas/password'as/skaičius...)
  • colspan - column span - lentelės laukelio plėtimas stulpeliais
  • target - anchor target - nurodo kur bus atidaryta nuoroda
