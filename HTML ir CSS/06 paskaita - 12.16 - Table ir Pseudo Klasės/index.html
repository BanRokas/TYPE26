  • tfoot - table foot - lentelės duomenų išvestinės (sumos / vidurkiai / ...)

  • th - table heading - lentelės stulpelio duomenų pavadinimas
  • td - table data - lentelės duomuo
  • caption - lentelės pavadinimas
  • colgroup - collumn group - stulpelių grupė

Attribute Selectors

  • [attribute] - atributo vardas
  • [attribute="reikšmė"] - atributo vardas ir tiksli reikšmė
  • [attribute~="reikšmė"] - atributo vardas ir reikšmės žodis
  • [attribute*="reikšmė"] - atributo vardas ir tokia pati reikšmės simbolių seka
  • [attribute^="reikšmė"] - atributo vardas ir reikšmė prasidedanti nurodyta seka
  • [attribute$="reikšmė"] - atributo vardas ir reikšmė pasibaigianti nurodyta seka
  • [attribute|="reikšmė"] - atirbuto vardas ir reikšmė prasidedanti nurodyta reikšmė arba reikšmė-
