Git ir Github

Git - programa skirta versijuoti kodą.

GitHub - svetainė, kurioje keliamas (versijuotas) kodas.

Git setup

  1. Įsirašyti Git programą
  2. Pasitikrinti ar tikrai įrašytas Git'as. Terminale panaudojant komandą "git -v".
  3. Susikonfigūruoti git'ą globaliai savo įrenginyje
    • git config --global user.name "GithubUserName"
    • git config --global user.email "GithubEmail"
    • git config --global core.editor "code"
    • git config --global core.autocrlf true|input (true jeigu windows | input jeigu mac/linux)
  4. Pasitikrinti ar teisingai atlika konfigūracija. Terminale panaudoti komandą "git config --global --edit" / Jeigu neveikia komanda - galima nueiti rankiniu būdu iki failo: "C:/Users/Tavo_PC_Vartotojas/.gitconfig"
