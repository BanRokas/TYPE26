Git ir Github

Git - programa skirta versijuoti kodą.

GitHub - svetainė, kurioje keliamas (versijuotas) kodas.

Git setup

  1. Įsirašyti Git programą
  2. Pasitikrinti ar tikrai įrašytas Git'as. Terminale panaudojant komandą "git -v".
  3. Susikonfigūruoti git'ą globaliai savo įrenginyje
    • git config --global user.name "GithubUserName"
    • git config --global user.email "GithubEmail"
    • git config --global core.editor "code"
    • git config --global core.autocrlf true|input (true jeigu windows | input jeigu mac/linux)
  4. Pasitikrinti ar teisingai atlika konfigūracija. Terminale panaudoti komandą "git config --global --edit" / Jeigu neveikia komanda - galima nueiti rankiniu būdu iki failo: "C:/Users/Tavo_PC_Vartotojas/.gitconfig"

Libraries ir Frameworks

Libraries - kažkieno kito parašytas kodas (kuris nėra originalioje kalboje), kuriuo galite naudotis savo projektuose.
Norint naudotis library kodu, reikia tą library prisijungti prie savo projekto panaudojant CDN.
Daugelis library veikia tokiu principu, kad jų viduje yra aprašytas kodas, kuriuo tu pats naudojiesi.
CSS libraries gali būti ClassLess, kurios stilius suteikinėja tag'ams.

Frameworks - kažkieno kito sukurta programa/programų rašymo būdas, kuriuo naudojantis patogiau kurti programas.

Preprocessors

CSS with superpowers

Suteikia galimybę rašyti CSS kodą naujoviškesniu būdu. Jame lengvai ir patogiai galima kurti kintamuosius, nest'inti CSS kodą, rašyti: sąlgas, funkcijas, ciklus, mix'inus ir t.t.

Naudojant betkurį CSS preprocesorių, reikia parašytą kodą konvertuoti (kompiliuoti) į paprastą CSS'ą, kurį jau prijungiame prie svetainės.

Units

Units - matavimo vienetai, skirti nustatyti elementų dydžius: plotį, aukštį, teksto dydį, tarpus, atitraukimus...

  • px - pikselis
  • % - pagal tėvinio elemento dydį
  • vw - viewport width - matomos svetainės plotis (100 - visas plotis)
  • vh - viewport height - matomas svetainės aukštis (100 - visas aukštis)
  • rem - root em(ephemeral) - 16px - pagal šakninį nustatymą
  • em - ephemeral - 16px - pagal tėvinio elemento nustatymą
  • ...
