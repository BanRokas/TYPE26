  • section - sekcija skirta kažkokiai temai aprašyti
  • article - straipsnis skirtas kažkokiai temai aprašyti
  • div - divas skirtas sukurti blokelį (grupuoti tag'us)
  • nav - navigation - navigacija

CSS Įvadas

CSS rašymo būdai

  • Inline - linijinis - kai CSS kodas rašomas tag'o atribute
  • Internal - vidinis - kai CSS kodas rašomas HTML faile, head tag'o -> style tag'o viduje
  • External - išorinis - kai CSS kodas rašomas atskirame CSS faile, kuris yra prijungiamas prie HTML naudojant link tag'ą

Inline

Linijinis CSS rašymo būdas yra naudojamas norint pasitestuoti kažkokius stilius.
Norint panaudoti linijinį CSS rašymo būdą, tag'ui reikia suteikti style atributą, kurio viduje rašome stilius.

Internal

Vidinis CSS rašymo būdas yra naudojamas norint pasitestuoti kokius nors selektorius ir stilius arba esant labai mažam HTML failui.
Norint panaudoti vidinį CSS rašymo būdą, reikia susikurti style tag'ą (kuris įprastai rašomas head tag'o viduje) ir jame aprašyti selektorius su stiliais.

External

Išorinis CSS rašymo būdas yra naudojamas dažniausiai.
Norint panaudoti išorinį CSS rašymo būdą, reikia susikurti naują CSS failą, kurį reikia prijungti prie HTML naudojant link tag'ą. O pačiame CSS faile rašyti įprastą CSS kodą su selektoriais ir stiliais.

Selektoriai

Selektoriai naudojami tam, kad CSS rašomi stiliai galėtų būti priskirti tam tikriems HTML elementams.

  • tag'o selektorius - nurodomas tag'o pavadinimas
  • multiple - per kablelį galima išrašyti keletą selektorių
  • * - visko selektorius
  • .class - susigalvojame klasės pavadinimą patys
  • #id - susigalvojame id pavadinimą patys

Class

Aprašant klasę CSS'e pradedame nuo to, kad parašome tašką ".", kuris indikuoja CSS'ui, kad bus kuriame klasė. Tuomet po taško rašome susigalvotą pavadinimą (be tarpų ir lietuviškų raidžių).
Norint priskirti klasę HTML'e, reikia elementui suteikti atributą "class" ir nurodyti norimos klasės pavadinimą.
Vienam elementui gali būti priskirta daug klasių ir viena klasė gali būti naudojama daug kartų. (many to many santykis)

ID

Aprašant ID CSS'e pradedame nuo to, kad parašome groteles "#", kurios indikuoja CSS'ui, kad bus kuriamas ID. Tuomet po grotelių rašome susigalvotą pavadinimą (be tarpų ir lietuviškų).
Norint priskirti ID HTML'e, reikia elementui suteikti "id" atributą ir nurodyti norimo ID pavadinimą.
Vienam elementui galima priskirti TIK vieną ID ir vieną ID reikėtų prisikirti TIK vienam elementui. (one to one santykis)
