Transform

Transform yra stilius, kuris nusako kad elementas bus kažkaip pakeičiamas, o visi žemiau išvardinti dalykai yra šio stiliaus nustatymai.

  • rotate(Z) - sukasi aplink Z ašį
  • rotateZ(Z) - sukasi aplink Z ašį
  • rotateX(X) - sukasi aplink X ašį
  • rotateY(Y) - sukasi aplink Y ašį
  • translateX(X) - pastūmimas X ašyje
  • translateY(Y) - pastūmimas Y ašyje
  • translate(X, Y) - pastūmimas X ir Y ašyse
  • scaleX(X) - keičia elemento dydį X ašyje
  • scaleY(Y) - keičia elemento dydį Y ašyje
  • scale(X, Y) - keičia elemento dydį X ir Y ašyse. Parašius vieną argumentą - jis taikomas abiejoms ašims.
  • skewX(X) - ištempiami X ašies kaštinėse esantys elemento kraštai.
  • skewY(Y) - ištempiami Y ašies kaštinėse esantys elemento kraštai.
  • skew(X, Y) - ištempiami X ir Y ašių kaštinėse esantys elemento kraštai.

Combined

Rotate, Translate ir Scale nustatymai jau turi savo atskirus stilius ir tam, kad juos panaudotume, nebebūtina jų rašyti kaip funkcijų transform stiliaus viduje.

Transform-origin

Transform origin nusako, nuo kurios elemento vietos vyksta rotate, scale ir skew kitimai.
Transform-origin default yra elemento centre.
