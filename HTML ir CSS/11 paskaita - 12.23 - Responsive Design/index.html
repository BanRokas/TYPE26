Breakpoints

Breakpoint - ekrano lūžis - kai iš vieno išdėstymo, pavyzdžiui telefoninio dydžio, yra pakeičiamas elementų išdėstymas į kitą išdėstymą, pavyzdžiui plančetinio dydžio.
Breakpoint'ų galima prikurti kiek tik nori (dažniausiai 3), kokiuose tiktais nori taškuose (standartų realiai nėra).

  1. x-small - portrait phones - 0px => 575px
  2. small - landscape phones - 576px => 767px
  3. medium - tablets - 768px => 991px
  4. large - laptop - 992px => 1199px
  5. x-large - desktop - 1200px => 1400px
  6. xx-large - large desktop - 1400px => infinity

Tam, kad būtų sukurtas vadinamasis breakpoint'as CSS'e reikia aprašyti jo media query.
Media query apsirašo tokiu būdu: @media (min-width: number_px){ ... css ... }

Media query nesuteikia jokių papildomų stiliaus prioritetų, dėl to media query reikia rašyti teisingu eiliškumu.
Media query stilai nepradingsta, pasiekus naują media query.

Mobile First

Mobile First dizaino principas yra tas, kad pradžioje puslapis yra pritaikomas mobiliesiems, o tik po to judama prie kitų ekrano dydžių.
Taip yra daroma dėl įvairių priežasčių: daugiau vartotojų naudoja mobiliuosius naršant internete; mažiau vietos informacijai - reikia atvaizduoti svarbiausią; lengviau pridėti turinį, negu atimti...

Rašant mobile first media query, būtina naudoti min-width, ne max-width nustatymą.
